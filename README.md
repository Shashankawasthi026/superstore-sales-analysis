# Superstore Sales Analysis Dashboard

## Project Overview
This project analyzes Superstore sales data using Excel, SQL, and Power BI to identify sales trends, profitability patterns, customer behavior, and actionable business insights.

## Tools Used
- Microsoft Excel
- MySQL
- Power BI
- SQL
- PDF Reporting

## Project Workflow
1. Data Cleaning in Excel
   - Removed duplicates
   - Handled missing values
   - Standardized text formatting
   - Fixed date and numeric formats

2. Exploratory Data Analysis
   - Sales by Region
   - Profit by Category
   - Monthly Sales Trend
   - Top Customers
   - Top Products

3. SQL Analysis
   Queries performed:
   - Sales by Region
   - Profitability by Category
   - Monthly Sales Trend
   - Discount Impact Analysis

4. Power BI Dashboard
   Dashboard includes:
   - KPI Cards
   - Regional Sales Analysis
   - Profit Category Analysis
   - Monthly Trend
   - Customer Analysis
   - Product Analysis
   - Interactive Filters

## Key Insights
- West region generated the highest sales.
- Technology category was the most profitable.
- High discounts significantly reduced profitability.
- Sales showed consistent growth over time.
- Top customers contributed significantly to total revenue.
